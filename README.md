# WakeOnLan

## Packet

대상 PC MAC address : 34-F2-22-12-12-12
                      
0xFF 0xFF 0xFF 0xFF 0xFF 0xFF 

0x34 0xF2 0x22 0x12 0x12 0x12       : 16 번 반복

UDP broadcast 전송.
